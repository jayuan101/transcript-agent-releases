# Transcript Agent

AI-powered transcription app for interviews and meetings. Automatically identifies speakers, generates formatted reports, and exports to Word or PDF.

## Download

Go to the [Releases](https://github.com/jayuan101/transcript-agent-releases/releases) page and download the latest `TranscriptAgent.exe`.

## How to use

1. Double-click `TranscriptAgent.exe`
2. The app opens in your browser at `http://localhost:7860`
3. Upload an audio or video file
4. Optionally enter the number of speakers (or leave blank to auto-detect)
5. Click **Transcribe**

## Supported formats

Audio: `.mp3`, `.wav`, `.m4a`, `.ogg`, `.flac`  
Video: `.mp4`, `.mkv`, `.webm`, `.mov`

## System requirements

- Windows 10 or 11 (64-bit)
- 8GB RAM minimum (16GB recommended)
- 5GB free disk space
